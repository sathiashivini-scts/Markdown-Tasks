# Student Management Application README

---

## Project Overview

Student Management Application is a simple system designed to store and manage student information. It allows users to add, view, update, and remove student records efficiently.

---

## Features

* Add new student details
* View student records
* Update student information
* Delete student records
* Search students by ID or name

---

## Installation Guide

1. Clone the repository.
2. Navigate to the project directory.
3. Install the required dependencies.


```
git clone https://github.com/sathiashivini-scts/Markdown-Tasks.git
cd Markdown-Tasks
pip install -r requirements.txt
```

---

## Usage Instructions

Run the application using the following command:

```
python app.py
```

---

## Sample Commands

```
python app.py --add "Student Details"

python app.py --search "Student ID"
```
